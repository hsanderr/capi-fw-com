Starting point of the program, where components are initialized and started, if applicable. 

The following operations are performed in this function\+:
\begin{DoxyItemize}
\item Non-\/volatile storage (NVS) is initialized.
\item Data is read from NVS (authorized MAC address).
\item Wi-\/\+Fi is initialized.
\item GPIOs are initialized.
\item Blue LED is blinked to indicate that the program has started.
\item VCC measurement is initialized.
\item Status component is initialized.
\item PWM component is initialized.
\item Beacon component is initialized.
\end{DoxyItemize}
